# Time Complexity

| n less than or equal to | Possible Complexities        |
| ----------------------- | ---------------------------- |
| $$10$$                  | $$O(n!), O(n^7),O(n^6)$$     |
| $$20$$                  | $$O(2^n*n), O(n^5)$$         |
| $$80$$                  | $$O(n^4)$$                   |
| $$400$$                 | $$O(n^3)$$                   |
| $$7500$$                | $$O(n^2)$$                   |
| $$7 * 10^4$$            | $$O(n\sqrt n)$$              |
| $$5 * 10^5$$            | $$O(nlogn)$$                 |
| $$5 * 10^6$$            | $$O(n)$$                     |
| $$10^{18}$$             | $$O(log^2n), O(logn), O(1)$$ |
